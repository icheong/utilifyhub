Color Format Reference Guide

Modern web design requires fluency across multiple color notations. Each format encodes the same visible color through a different mathematical model.

HEX

The most common CSS format: a hash followed by 6 hexadecimal digits (e.g. #6366f1). Each pair ranges from 00 to ff.

RGB

Defines color as three integers from 0–255. Used in CSS as rgb(99, 102, 241).

HSL

Hue is a degree on the color wheel (0–360), Saturation is intensity (0–100%), and Lightness ranges from black (0%) to white (100%).

HSB / HSV

Similar to HSL but replaces Lightness with Brightness. Widely used in design applications like Photoshop and Figma.

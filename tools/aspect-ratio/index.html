What is an Aspect Ratio?

An aspect ratio represents the proportional relationship between the width and height of an image, video container, or monitor screen. Written as two values separated by a colon (e.g. `Width:Height`), it indicates scale dimensions regardless of overall screen resolutions.

Common Aspect Ratios & Use Cases

  • 16:9 (High Definition): The international standard format for HDTVs, digital screens, YouTube streams, and standard monitors. Common resolutions include 1920x1080 (FHD) and 3840x2160 (4K).
  • 9:16 (Vertical HD): The exact inversion of 16:9. Heavily utilized across smartphone layouts, TikTok clips, Instagram Reels, and YouTube Shorts.
  • 4:3 (Academy Standard): The vintage TV standard preceding HD formats, occasionally utilized in retro photography and design workflows.
  • 1:1 (Square Format): Equal width and height dimensions, popularized by social platforms (Instagram posts) and profile avatar dimensions.
  • 21:9 (Ultrawide Cinemascope): Wider widescreen layout common in gaming monitors and cinematic movie projections.

The Mathematics Behind Aspect Ratio Sizing

To reduce dimensions to their simplest aspect ratio, we compute the Greatest Common Divisor (GCD) of the width and height. Dividing both width and height by the GCD yields the simplified aspect ratio.

Simplified Width = Width / GCD
Simplified Height = Height / GCD

Proportional scaling follows direct mathematical ratios. For example, maintaining an aspect ratio (W1/H1) implies:
New Height = (New Width * H1) / W1
New Width = (New Height * W1) / H1
